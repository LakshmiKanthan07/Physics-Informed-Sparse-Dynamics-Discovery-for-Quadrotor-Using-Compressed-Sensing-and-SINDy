# Code of Conduct

## Our Pledge

We as contributors pledge to make participation in the Quadrotor SINDy project a harassment-free experience for everyone.

## Our Standards

- Be inclusive and respectful.
- Use welcoming language.
- Show empathy and patience.

## Reporting

If you experience or witness unacceptable behavior, please open an issue or contact the repository maintainer.

## Scope

This code of conduct applies to all project spaces, including issue trackers, pull requests, and project chats.
